# Proyecto Astro + Tailwind CSS

Este proyecto ha sido creado con Astro y Tailwind CSS usando Vite como base.

## Scripts principales
- `npm run dev`: Inicia el servidor de desarrollo.
- `npm run build`: Genera la versión de producción.
- `npm run preview`: Previsualiza la versión de producción.

## Estructura
- `src/` - Código fuente del sitio.
- `src/styles/global.css` - Estilos globales con Tailwind.

## Notas
- Puedes modificar el contenido inicial en `src/pages` y los estilos en `src/styles/global.css`.
